## Welcome to Dathost’s unofficial python wrapper!
This is a unofficial asynchronous & synchronous wrapper for Dathost’s API.

### Features:
- Full API coverage.
- Background authentication.
- Asynchronous & synchronous support.
- Easy to use with an object oriented design.
- 100% test coverage.
- Upload URL caching.


### Install
- Pip: ``pip3 install backblaze``
- Git: ``pip3 install git+https://github.com/WardPearce/backblaze.git``
